# NOTES

MAKE SURE YOU MAKE A NEW ACCOUNT AND FOLLOW ALL OF THE BELOW NOTES. IF YOU CHOOSE NOT TO, YOUR REFUND RIGHTS WILL BE VOIDED.&#x20;

* This applies to all games: Do not EVER login to Microsoft, OneDrive. Do not EVER install the following apps: SteelSeries, Logitech G Hub, Razer, NVIDIA App, GeForce Experience and any similar apps that interact with your peripherals/hardware. They are all potential flags. A substitute for Logitech G Hub is the [Onboard Memory Manager](https://download01.logi.com/web/ftp/pub/techsupport/gaming/OnboardMemoryManager_2.5.358.exe).
* If you are on an Intel CPU, please follow [VHD](/blitz-perm/intel-disk-alternative/vhd.md) and install the game onto the VHD.
* If you are reinstalling Windows with a USB drive after you have spoofed and you have an ASUS motherboard, be sure to AVOID deleting the EFI partition that the "autopartition.bat" made. This way, the spoof will stay even after reinstalling Windows (Just make sure UEFI OS is still #1 in the boot order though!)
* If you use LAN (Ethernet), don't install WLAN (WiFi) or Bluetooth drivers. You are able to install chipset drivers, GPU drivers (For [NVIDIA GPUs](https://www.nvidia.com/en-us/drivers/), select the option without the NVIDIA app when running the driver installer), NIC (LAN) drivers.
* If you are spoofing for any EAC (Fortnite, Rust, Apex) game, make sure you look at [VPN](/blitz-perm/fortnite-rust-apex-eac/vpn.md) and [TPM](/blitz-perm/valorant-lol-fn-tourneys-cod-bo7/tpm.md), and that you do not have any external drives/USB drives plugged in while playing the game (these can be flags).&#x20;
* If you are spoofing for any VGK (VALORANT, LOL) game, make sure you look at [TPM](/blitz-perm/valorant-lol-fn-tourneys-cod-bo7/tpm.md) and you do not have any external drives/USB drives plugged in while playing the game. We also recommend upgrading to 25H2, as it seems to remove certain flags. The new account is CRUCIAL.
* As an important precaution FOR ALL GAMES, after spoofing, we require you to play on a NEW account that you make after spoofing until you've reached AT LEAST 24 hours of total playtime in the game, and we require you to not to have any DMA card or aim devices plugged in + no cheats for this 24 hours of playtime. After you have completed this playtime requirement, you can then switch to another account, plug in your DMA card, use a cheat, etc. **This is NOT a recommendation, it is a requirement in order to remove as many external factors as possible in order to single out if the spoof was the problem (if you are delayed).**&#x20;

{% hint style="info" %}
After you have read, followed and understood all of these notes, you are able to install the game, assuming staff have given you the go-ahead.
